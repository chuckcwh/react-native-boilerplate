# CocoaPods

Use CocoaPods for managing the ios dependencies

---

### Installation 
```
sudo gem install cocoapods -v 1.5.3
```

---

### Version
CocoaPod: 1.5.3

--- 

### Initialize CocoaPod Project
```bash
cd ios && pod init 

# after hitting the command above, edit Podfile to remove TvOS setting
vi Podfile

// install Pod projec 
pod install
```
